FavToDoTab

A simple intuitive but efficient ToDo App developed using jQuery and jQuery-ui.

Demo Fiddle

Features : (_More features to be added on Future Updates_)

  • Tabbed Panes | Task-Tab and Favorites-Tab - Tasks can be easily stored on Favorites Tab by clicking on the star icon (icon gets changed to filled star - so as to show distinguish that it's listed on favorites tab). - Stored Favorites Tasks can be listed back to Tasks Tab by clicking on the home icon.

  • Duplicacy Check - Checks for duplicate tasks on Task Tab as well as on Favorites Tab
    --- UI-UX Features :
  • Add Button gets enabled only when there is some text in the textbox (_also detect blank spaces_)
  • Other than clicking on Add Task button, tasks can be added by hitting enter key
